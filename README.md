# twitter_clone
This project is a full-stack Twitter clone developed using MongoDB, Express, Vanilla JavaScript, Node.js. The application features user account creation, login with cookie authentication, tweet creation, following/unfollowing functionality, a following feed, image uploads, liking tweets, and dynamic profile management. It is designed to provide a streamlined user experience and is hosted using Docker on DigitalOcean for scalable deployment.
## Functionality:
-User Authentication: Sign up, login, and logout with secure cookie-based authentication.
-Profile Management: Edit profiles, upload profile images, and update user bios.
-Tweet Functionality: Create and delete tweets, upload images with tweets, and like tweets.
-Follow System: Follow and unfollow users to customize your following feed.
-Following Feed: See tweets from the users you follow in real-time.
-Responsive Design: Adapted for various screen sizes to ensure a great user experience on desktop and mobile devices.
## Technologies Used
-Frontend: Vanilla JS, HTML, CSS, JavaScript
-Backend: Node.js, Express.js, MongoDB
-Database: MongoDB (NoSQL database for storing user data and tweets)
-Hosting: Hosted on DigitalOcean
-Authentication: Cookie-based user sessions for secure authentication
-Image Uploads: Integrated image upload functionality using Multer.
